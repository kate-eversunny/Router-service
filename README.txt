Проект представляет собой микросервис, способный определять, есть ли прямой маршрут между двумя остановками на основе информации,
полученной из файла маршрутов.

Пример файла маршрутов, где первое число в каждой строке обозначает номер маршрута,
а остальные числа - номера остановок:
0 0 1 2 3 4
1 3 1 6 5
2 0 6 4

Запустить проект можно с помощью maven:
mvn spring-boot:run -Dspring-boot.run.arguments=--routeFilePath={путь_к_файлу_маршрутов}

Пример запроса к сервису:
http://localhost:8080/api/direct?from=3&to=6
